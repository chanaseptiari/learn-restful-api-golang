# Learning Restful API

Restful api build with golang v1.19

## Compile Project

```bash
GOOS=linux GOARCH=amd64 go build -ldflags="-w -s" -o build/learn-restful-api-golang
```

## Running Project

```bash
go run main.go
```

OR

```bash
GOOS=linux GOARCH=amd64 go build -ldflags="-w -s" -o build/learn-restful-api-golang
./build/learn-restful-api-golang
```
